# Скрипт для создания карточек лекарств
*** 
## Назначение 
##### Данный скрипт помогает создавать готовые карточки лекарств с логотипом и ценой.
Программа написана на **Python** с применением библиотеки **Pillow**

## Установка зависимостей

Первым делом нужно установить Python с официального сайта, если у вас его нет. Проверить, установлен он или нет, можно в терминале введя команду:

>python --version


#### Далее вводим в терминал цепочку команд:
1. git clone `https://github.com/nhamaganov/photo_changer.git`
2. cd photo_changer
3. python -m venv venv
4. .\venv\Scripts\activate
5. pip install -r requirements.txt

## Применение

1. Подготавливаем изображение лекарства, желательно положить его в один каталог вместе со скриптом.

2. Убедиться, что внутри терминала вы находитесь в одном каталоге вместе с исполняемым файлом

3. Далее воспользоваться командой в терминале:


> python main\.py <путь_к_изображению> <цена> [имя_итогового_файла] 

где `путь_к_изображению` и `цена` - обязательные аргументы, а `имя_итогового_файла` - опциональный

После данной команды итоговое изображение сохранится внутри каталога, в котором находится исполняемый файл.

Если не указать имя итогового файла, то изображение сохранится как **result.png**
***
#### Примеры использования:
* py main\.py pills/pill3.webp 912.00 final_img.png
* py main\.py my/catalog/of/pills/pill3.webp 124.36

> Важно отметить, что путь к изображению пишется относительно родительского каталога, иначе если файл изображения лежит в отдельном месте, лучше использовать абсолютный путь

* py main\.py "C:\Users\MyPC\Desktop\Learning\pharmImg.webp" 912.00
