# Скрипт для создания карточек лекарств
*** 
## Назначение 
##### Данный скрипт помогает создавать готовые карточки лекарств с логотипом и ценой.
Программа написана на **Python** с применением библиотеки **Pillow**

## Установка зависимостей

#### Ввести в терминал цепочку команд:
+ python -m venv venv
+ .\venv\Scripts\activate
+ pip install -r requirements.txt

## Применение

1. Для начала нужно подготовить изображение лекарства, желательно положить его в один каталог вместе со скриптом.

2. Далее воспользуемся терминалом:
+ python main.py <путь_к_изображению> <цена> [имя_готового_файла]
